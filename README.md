[[독일]] [[튀링겐]] 주에 위치해 있는 공립 연구중심 공과대학이다.  [[튀링겐]] 주의 유일한 공과대학이며 5개의 학부로 구성되어 있다.
19개의 학사 과정, 25개의 석사 학위 과정, 11개의 복수 학위 과정을 제공한다.
== 역사 ==
1894년 튀링겐 사립 기술훈련대학(Thüringisches Technikum)으로 시작하였다. 
이후 전자기술대학(Hochschule für Elektrotechnik, HfE)과 기술대학을 거쳐 1992년 공과대학교로 승격됐다.

== 국제교류 ==
전 세계 여러 대학과 교환 프로그램을 운영하는 등 국제 교류 및 협력을 이어가고 있다.

일메나우공과대학교와 국제 교류 협정을 체결한 한국 내 대학은 [[파일:명지대학교 엠블럼.svg|width=18px]] [[명지대학교]] 한 곳 뿐이다.
